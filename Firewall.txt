

#Firewall is security program, which is secure from outside network 
it is controlling inbound and outbound and monitoring 
#there are two type firewall - Hardware / Software.

Netfiltering - it allow the kernel module to inspect every packate 

There are nine zone
Predefined zone
1. Ttrusted - Allow all incoming traffic
2. Home - reject incoming traffic unless related to outgoing traffic
3.internel - same as home zones
4.work - reject infomming traffic unless related to outgoing traffic 
5. public zone - ssh dhcp default
6.externel - it is working ssh
7.DMZ - Reject incoming traffic 
8.block - reject incomming outgoing traffic
9. Drop - drop all incomming trafic
